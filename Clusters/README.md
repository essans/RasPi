### Things to remember when using cluster_config.py


Chaining commands:

```
./...py -c 'cd code/python && sudo cp /etc/hosts test_hosts'

```

```
./...py -c 'cd code && sudo mkdir python'

./....py -c 'sudo chmod -R 0777 code'   #full permissions


```
